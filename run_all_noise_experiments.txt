# ============================================================================
# NOISE INJECTION INTERPRETABILITY EXPERIMENTS
# ============================================================================
# For each curriculum stage, train with Gaussian noise vs real signals,
# then evaluate with signal contribution tracking to test whether the
# model actually uses the time series signal content.
#
# Three conditions per stage:
#   1. real_<task>         — real signals, real stats in text
#   2. noise_<task>        — noise signals, real stats kept in text (can the model cheat via stats?)
#   3. noise_stripped_<task> — noise signals, stats stripped from text (no shortcut possible)
#
# Each stage starts from the checkpoint 1 stage before:
#   - Stage 1 (TSQA): fresh model (no predecessor needed)
#   - Stage 3 (HAR-CoT): stage 2 (captioning) checkpoint
#   - Stage 4 (Sleep-CoT): stage 3 (HAR-CoT) checkpoint
#   - Stage 5 (ECG-CoT): stage 4 (Sleep-CoT) checkpoint
#
# IMPORTANT: Each stage uses its own experiment name to avoid directory collisions.
# ============================================================================

# ============================================================================
# PREREQUISITES: Download intermediate HF checkpoints

    --model OpenTSLMFlamingo \

    --noise_seed 67 \
